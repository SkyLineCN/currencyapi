# 	通用API项目

​																																									*作者：skyline*



​		个人写的小玩意，用来处理通用的API接口。短期目标是通过表来控制接口的读取和操作。纯后端接口代码，测试通过postman进行。目前设想有以下几部分：

​	1，使用者授权ID。

​		  每个调用API的有一个唯一ID。只有输入ID才允许调用接口。

​	2，使用者接口授权。	

​		  具体为：使用者对接口有没有put，get等权限。		

​	3，使用者日志记录。

​           谁在什么时候调用了什么接口，json数据是什么？需要记录到日志表中。

短期目标暂定这三个。
